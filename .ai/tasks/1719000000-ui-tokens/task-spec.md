Define `@keyframes` blocks below the `@theme inline` block (standard CSS) for the animations currently inlined in page components:

2. **`wave-1`** through **`wave-5`** — from `speaker/page.tsx` inline `<style>`

Exact keyframe code from the source pages (preserve verbatim):

